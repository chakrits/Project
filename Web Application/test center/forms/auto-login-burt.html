How to Use BURT Auto Login Generator
Instructions:
  1. Configure target URL and application ID
  2. Enter secret key for encryption
  3. Toggle BURT parameters you want to include
  4. Fill in values for enabled parameters
  5. Use "Gen Timestamp" for current Unix timestamp
  6. Click "Generate BURT Login URL"
  7. Copy the generated URL or test it directly
BURT Features:
  • JSON Format: Parameters sent as JSON object
  • Unix Timestamp: Automatic timestamp generation
  • AES Encryption: Secure data transmission
  • URL Pattern: application_id and encrypted key
  • Client Authentication: Support for client_id
URL Pattern: https://www.testurl.com?application_id={{app_id}}&key={{encryption}}
