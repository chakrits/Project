How to Use Document List URL Generator
Instructions:
  1. กรอก Target URL, App ID, Secret Key
  2. เปิด/ปิด Parameters ที่ต้องการส่ง
  3. กรอกค่าใน Parameters ที่เปิดใช้งาน
  4. กด "Generate URL" เพื่อสร้าง Encrypted URL
  5. Copy หรือ Open URL ที่ได้
Security Notes:
  • AES Encryption: ข้อมูลถูก Encrypt ด้วย AES
  • JSON Payload: Parameters ถูกแปลงเป็น JSON ก่อน Encrypt
  • Local Processing: Encrypt ทำบน Browser ไม่ส่งขึ้น Server
  • DateTime: request_datetime +/- ได้ไม่เกิน 5 นาที
Important: URL ที่สร้างมีข้อมูล Authentication ที่ถูก Encrypt กรุณาเก็บรักษาและส่งต่ออย่างปลอดภัย
