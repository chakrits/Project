About AES Encryption

Advanced Encryption Standard (AES) is a symmetric encryption algorithm. AES is the industry standard as it allows 128 bit, 192 bit and 256 bits encryption. This tool provides multiple modes of encryption and decryption such as ECB, CBC, CTR and GCM mode.

Security Notice: Any secret key value that you enter is not stored on this site. All encryption/decryption is performed locally in your browser.

Key Features & Modes
Encryption Modes:
  • ECB: Electronic Code Book - Simple but less secure
  • CBC: Cipher Block Chaining - Recommended, requires IV
  • CTR: Counter Mode - Stream cipher, good for parallel processing
  • GCM: Galois/Counter Mode - Most secure with authentication
Key Features:
  • Industry standard encryption
  • 128, 192, 256-bit key sizes
  • Symmetric key algorithm
  • Fast and secure encryption
