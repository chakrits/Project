How to Use Auto Login Generator
Instructions:
  1. Load system configuration or enter manually
  2. Configure target URL, org code, and app ID
  3. Toggle parameters you want to include
  4. Fill in values for enabled parameters
  5. Click "Generate Auto Login URL"
  6. Copy the generated URL or test it directly
Security Notes:
  • AES Encryption: Uses secure AES encryption
  • Secret Key: Loaded from configuration
  • Local Processing: All encryption done client-side
  • DateTime Format: YYYY-MM-DDTHH:mm:ss (Local Time)
Important: The generated URL contains encrypted authentication data. Handle with care and ensure secure transmission.
